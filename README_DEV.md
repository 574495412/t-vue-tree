# 一个vue tree（同步树，没有异步树）的npm组件包开发项目


## 目录

```
├── dist/           (编译后的、发布用的 )
├── examples/        (使用示例)
├── node_modules/   (运行包)
├── src/            (开发用的目录 )
├── test/            (单元测试)
├── package.json  (项目描述)
└── webpack.config.js （webpack配置）
```

1.下载npm包:

    npm install t-vue-tree
